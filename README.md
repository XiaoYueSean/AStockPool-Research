# AStockPool-Research
本仓库是 **键盘财富说** 的股票研究池，旨在为投资者和研究人员提供一个系统化的股票研究平台。仓库中包含经过筛选和分析的A股股票，以及相关的研究资料和分析代码。

## 目录结构
- **Stock_Lists/**：包含不同主题和策略的股票池列表。
- **Research_Notes/**：存放关于每只股票的研究笔记和分析报告。
